# Final Project - Emotion Detector

An AI-based web application that detects emotions from text using IBM Watson NLP.

## Project Description
This is the Final Project for the course. The application uses Watson NLP to analyze text and detect emotions including anger, disgust, fear, joy, and sadness.

## Features
- Emotion detection using Watson NLP
- Flask web interface
- Error handling for invalid input
- Unit tests
- Static code analysis (Pylint)
